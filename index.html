O que é GitHub

quarta-feira, 6 de março de 2024

O GitHub é uma plataforma online usada para hospedar e gerenciar projetos de código-fonte. Imagine-o como um depósito virtual onde você guarda o código do seu software, podendo controlar versões, colaborar com outros desenvolvedores e acompanhar o histórico de alterações. É como um drive virtual específico para projetos de programação, mas com funcionalidades extras que facilitam o trabalho em equipe e o controle do código.

O que é GitHub Pages

quarta-feira, 6 de março de 2024

GitHub Pages é um serviço oferecido pelo GitHub que permite a publicação de sites estáticos diretamente do seu repositório. Isso significa que você pode criar um website utilizando HTML, CSS e JavaScript e hospedá-lo gratuitamente no GitHub.
